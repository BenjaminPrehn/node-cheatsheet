Setup Nodemon

Nodemon is a usefull tool when developing on a node server. Nodemon automatically restart your node server whenever you make changes in the code and thereby you don't have to re-run it manually each time.

Documentation on Nodemon can be found here

Install Nodemon

If you never used Nodemon before you need to install it. Installation can be done trough terminal if you already have Node.js installed on your computer.

To install it you just need to run the below command in your terminal:

Terminal Command: 

                        
npm install -g nodemon

When it have been succesfully installed, then you are ready to start using nodemon

How to use Nodemon?

Nodemon is super simple to use. You just have to replace "node" with "nodemon" when running your node server.

Terminal Command: 

                        
nodemon app.js

The above command will start your server with nodemon, and it will therefore automatically re-run your code when any changes accure.

The terminal should respond something like this if your node server is running succesfully.

                        
[nodemon] 2.0.7
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
[nodemon] starting `node app.js`
Server is running on port 8080

You can always restart the server manually by writting "rs" into the terminal!
