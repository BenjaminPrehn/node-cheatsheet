How to setup routes

Within our node server, we want to be able to create new routes - So a user can navigate to new pages on our server or if we would want to make an API we need to specify diffrent routes for the endpoint the user needs to use to navigate the API.

Inside our app.js file we can therefore set up new routes - So if the user navigate to http://localhost:8080/routes they would see information on how to setup a route.

app.js: 

                    
app.get("/welcome", (req, res) => {
    res.send("Welcome");
});

if you would like to send back some html when a user visit the route you can do so by the following code:

app.js: 

                 
app.get("/routes", (req, res) => {
    res.sendFile(__dirname + "/public/routes.html");
});

With the above code you'll send the html file to the route that you specified
