Get started with Node.js

To get started with your very first Node.js server you'll need to Node.js webiste and download the correct version for your computer.

After it have been succesfully downloaded you'll be able to create your first project. Start out by creating a new folder in a location that is easily accesible by you.

Then you need to create two files named: app.js and package.json - When both files have been created - then navigate inside package.json. Inside package.json we will have to setup up the dependencies we would have to use. For this example we will setup "express".

File: package.json

                            
{
    "dependencies": {
        "express": "^4.17.1"
    }
}

After you have inserted the above code into your package.json file, then you'll be ready to open your terminal and navigate inside the current project. When you have made sure you are at the right place - Then you can insert the following line into the commandline, to start installing the node modules needed and specified for this project.

Terminal command:

                        
npm install

Now you should notice that the install generated a new folder called "node_modules" and a file called "package-lock.json" - If so, then you are on the right track.

Now you can open up your app.js file - and start inserting the below code to get your node.js server up and running.

File: app.js:

                        
const express = require('express');
const app = express();

app.listen(8080, (error) => {
    if (error){
        console.log(error);
    }
    console.log("Server is running on port", 8080)
});

When the above code have been inserted into app.js file, you'll need to run the below terminal command to start up your node.js server.

Terminal Command

              
node app.js

The server will now start on port 8080, since we told the app to listen on that port.

This is how you would access the server. http://localhost:8080

Now you made your very first node server - Congratualtions!
